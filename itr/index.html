ITR

Image Tools Resource (ITR) is an index of suggested open-source analysis software for bioimaging. Though a work-in-progress, the aim is to publish tools that have links to accessible datasets, benchmarking resources, and public workflows to aid in training and reproducibility.

Below you will find community-driven projects, their entry in the Bio Image Information Index (https://biii.eu), domains in which the software is applicable, tags sampling the tasks that the software is useful for, and a list of datasets, both in the IDR and elsewhere.

If you would like to submit tools to the ITR, please email us at itr@openmicroscopy.org.
