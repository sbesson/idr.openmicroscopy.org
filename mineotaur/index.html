Mineotaur

A tool for high-content microscopy screen sharing and visual analytics.

When the IDR was launched we included Mineotaur, a high content visual analytics tool for exploring genes controlling cell shape, microtubule organization, and cell-cycle progression in S. pombe.

We have temporarily removed Mineotaur whilst we rewrite it to work with all screens in the IDR. In the meantime you can find the source code and deployment instructions on GitHub.
